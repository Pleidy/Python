
PointSubtract
用一个点与另一个点或一个向量相减。
语法
rhinoscriptsyntax.PointSubtract (point1, point2)
rhinoscript.pointvector.PointSubtract (point1, point2)
参数




point1
必须参数。包含三个数字的列表或者 Point3d 对象。被减的 3-D 点。


point2
必须参数。数组。包含三个数字的列表或者 Point3d 对象。用于相减的 3-D 点或 3-D 向量。


返回值




Point3d
如果执行成功，返回计算得到的 3-D 点。


None
出错返回空值。


示例
import rhinoscriptsyntax as rs
point1 = (1,1,1)
point2 = (2,2,2)
point = rs.PointSubtract(point1, point2)
print point

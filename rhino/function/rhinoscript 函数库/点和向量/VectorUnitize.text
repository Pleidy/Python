
VectorUnitize
对一个 3-D 向量执行统一或标准化操作。注意，零向量无法标准化。
语法
rhinoscriptsyntax.VectorUnitize (vector)
rhinoscript.pointvector.VectorUnitize (vector)
参数




vector
必须参数。包含三个数字的列表或者 Vector3d 对象。要标准化的 3-D 向量。


返回值




Vector3d
如果执行成功，返回计算得到的 3-D 向量。


None
出错返回空值。


示例
import rhinoscriptsyntax as rs
vector = rs.VectorUnitize( [1.5,-4.1,3.6] )
print vector


VectorDivide
用 3-D 向量与一个数相除。
语法
rhinoscriptsyntax.VectorDivide ( vector, divide )
rhinoscript.pointvector.VectorDivide ( vector, divide )
参数




vector
必须参数。三个数字构成的列表或 Vector3d 对象。被除的 3-D 向量。


divide
必须参数。数字。非 0 的除数。


返回值




Vector3d
如果执行成功，返回计算得到的 3-D 向量。


None
出错返回空值。


示例
import rhinoscriptsyntax as rs
vector = rs.VectorDivide((5,5,0), 5)
print vector

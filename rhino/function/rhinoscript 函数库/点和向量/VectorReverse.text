
VectorReverse
反转一个 3-D 向量的方向。
语法
rhinoscriptsyntax.VectorReverse (vector)
rhinoscript.pointvector.VectorReverse (vector)
参数




vector
必须参数。包含三个数字的列表或者 Vector3d 对象。3-D 向量。


返回值




Vector3d
如果执行成功，返回计算得到的 3-D 向量。


None
出错返回空值。


示例
import rhinoscriptsyntax as rs
vector = rs.VectorReverse([1,0,0])
print vector


DefaultRenderer
返回或修改 Rhino 当前或默认的渲染器插件，使用 PlugIns 方法可以获得一个可用渲染插件的列表。
语法
rhinoscriptsyntax.DefaultRenderer( renderer=None )
rhinoscript.application.DefaultRenderer( renderer=None )
参数




renderer
可选参数。字符串。要设置为默认渲染器插件的名称。


返回值




字符串
如果没有指定渲染器，返回当前渲染器的名称。


字符串
如果指定了渲染器，返回先前渲染器的名称。


示例
import rhinoscriptsyntax as rs
rs.DefaultRenderer("MyRenderPlugIn")


UnlockObject
解锁一个已锁定的物件。锁定的物件可见，也可以被物件锁点捕捉。但是，不能被选择。
语法
rhinoscriptsyntax.UnlockObject (object_id)
rhinoscript.object.UnlockObject (object_id)
参数




object_id
必须参数。字符串或 Guid。要解锁物件的 ID 。


返回值




布尔值
True 或 False 表示计算完成或失败。


示例
import rhinoscriptsyntax as rs
obj = rs.GetObject("Select object to lock")
if obj: rs.LockObject(obj)
#Do something here...
rs.UnlockObject( obj )

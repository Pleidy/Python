
RenderAntialias
返回或设置渲染器抗锯齿。抗锯齿使得每个被一条以上光线照射的像素能显示的更好。
pixel.增加抗锯齿，将会增加渲染时间。详情请参考 Rhino 帮助文档中关于 文档属性 的部分。
语法
rhinoscriptsyntax.RenderAntialias (style=None)
rhinoscript.document.RenderAntialias (style=None)
参数




style
可选参数。数字。抗锯齿类型。




值
描述


0
None


1
正常且较慢。


2
最好且最慢。




返回值




数字
如果没有指定 style 参数，将返回当前的抗锯齿类型。


数字
如果指定了 style 参数，将返回先前的抗锯齿类型。


示例
import rhinoscriptsyntax as rs
rs.RenderAntialias(1)

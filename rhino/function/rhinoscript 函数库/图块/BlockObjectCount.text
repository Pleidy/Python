
BlockObjectCount
返回一个图块定义中包含物件的数量。
语法
rhinoscriptsyntax.BlockObjectCount (block_name)
rhinoscript.block.BlockObjectCount (block_name)
参数




block_name
必须参数。字符串或 Guid。现有图块定义的名称。


返回值




数字
图块定义中包含物件的数量。


示例
import rhinoscriptsyntax as rs
count = rs.BlockObjectCount()
print "There are", count, " blocks."

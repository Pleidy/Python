
ZoomExtents
将指定视图或当前活动视图缩放到可见物件的范围。
语法
rhinoscriptsyntax.ZoomExtents (view=None, all=False)
rhinoscript.view.ZoomExtents (view=None, all=False)
参数




view
可选参数。字符串或 Guid。视图的名称或 ID 。如果省略，将使用当前视图。


all
可选参数。布尔值。缩放所有的视图。如果省略，只缩放指定视图（False）。


返回值
None
示例
import rhinoscriptsyntax as rs
rs.ZoomExtents()
